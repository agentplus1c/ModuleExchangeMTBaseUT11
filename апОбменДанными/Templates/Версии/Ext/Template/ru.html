История версий обработки обмена данными

2.0.2.78 (06.05.19)

  • Исправлена выгрузка справочника "Номенклатура" с нулевыми остатками.
  • Исправления для конфигурации Управление торговлей, редакция 11.2 (11.2.3.95): 
    • Доработана загрузка документов  "Заказ клиента", "Реализация товаров и услуг", "Поступление товаров", "Приходный кассовый ордер" и "Расходный кассовый ордер".

2.0.2.75 (19.04.19)

  • Добавлен новый тип документов "Задание". Документ "Задание" предназначен для постановки сотруднику списка задач, которые ему необходимо выполнить в течение дня. Например, создание какого-либо документа, обмен данными и т.п. (Главная - Документы - Задание).
  • Добавлен новый справочник "Виды задач". Справочник предназначен для хранения списка задач, которые могут использоваться для документа "Задание". (Главная - Справочники - Виды задач).
  • В отчет "Выполнение плана посещений" добавлена настройка "Список торговых точек". При заполнении списка торговых точек, в отчете появляется возможность  проанализировать план/факт визитов сразу по нескольким маршрутам. Если настройка не заполнена, то отображаются только торговые точки, закрепленные за агентом (в настройках агента, поле "Маршрут").
  • Изменен контактный номер технической поддержки.

2.0.2.72 (10.04.19)

  • Добавлена настройка "Выгружать долги только по документам агента." Настройка доступна: "Обмен данными" - "Общие настройки" -  вкладка "Основные"
  • Добавлена дополнительная настройка проведения документов. Появилась возможность настроить список типов документов, которые при загрузке нужно проводить. Настройка доступна: "Обмен данными" - "Общие настройки" -  вкладка "Основные" - ссылка "настроить" напротив настройки "Проводить документы".

2.0.2.1  (28.05.18)

  • В настройках торговых агентов (форма открывается в разделе "Обмен данными", ссылка "Настройки торговых агентов")  в разделе "Выгружаемые данные" добавлен флажок "Только виды цен из соглашений". Если флажок выбран, то в МУ выгружаются только виды цен, указанные в выгружаемых соглашениях. Если одновременно с флажком "Только виды цен из соглашений" выбран флажок "Виды цен", то в выгрузку попадают виды цен, отвечающие обоим условиям одновременно (каждый выгруженный вид цены есть в списке "Виды цен" и есть среди выгружаемых соглашений). Если в учетной системе выключено использование соглашений, то значение флажка "Только виды цен из соглашений" игнорируется.
